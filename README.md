# Cron Monitor — 9acc Vex

Static GitHub Pages dashboard untuk dokumentasi dan monitoring LaunchAgent/cron Vex yang terkait 9acc.

- Browser tidak memanggil Meta API.
- Repo tidak menyimpan token/secret.
- Data snapshot di `data/cron.json` digenerate dari Mac Mini launchd state + log metadata.
- EVPO-73 controller khusus sengaja dikecualikan dari dashboard ini.

URL Pages: https://kedebuk.github.io/cron-monitor-9acc-vex/
